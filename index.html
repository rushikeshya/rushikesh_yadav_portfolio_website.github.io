My Interests

Data scientist

A data scientist is responsible for collecting, processing, and analyzing large and complex datasets using statistical and machine learning techniques. They use their findings to provide insights and help organizations make data-driven decisions.

Data Analyst

A data analyst is responsible for collecting, cleaning, and analyzing data to identify patterns and trends. They help organizations make informed decisions by presenting data in a way that is easy to understand.

Machine Learning Engineer

A machine learning engineer is responsible for designing, building, and deploying machine learning models. They work with data scientists to develop algorithms and deploy them in real-world applications.

SQL Developer

A SQL developer is responsible for designing, developing, and maintaining databases using SQL (Structured Query Language). They create and manage databases, write and optimize SQL queries, and ensure the security and integrity of the data.
